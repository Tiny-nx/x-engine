## preloadUniMP

 预加载后打开小程序

	
**参数说明**

| name                        | type      | optional | default   | comment  |
| --------------------------- | --------- | -------- | --------- |--------- |
| appId | string |  |  |  |
| arguments | Map\<string,string\> |  |  | 配置启动小程序时传递的参数 |
| redirectPath | string |  |  |  路径 |
| enableBackground | bool |  |  |  开启后台运行 |
| showAnimated | bool | true | true | 是否开启 show 小程序时的动画效果 默认：true |
| hideAnimated | bool | true | true | 是否开启 hide 时的动画效果 默认：true |
