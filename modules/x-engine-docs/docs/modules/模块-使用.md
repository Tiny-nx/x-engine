# 模块接入指引

## 手动

各项目主工程里按模块引入即可。

比如 android  用　gradle。

iOS　用　Podfile。　路径为本地相对路径。



## 自动（开发中）

以蓝牙模块为例. 

1. 找到相应的模块. 如  @zkty-team/x-engine-module-ui

3. 安装相应模块
   ```
   npm install @zkty-team/x-engine-module-ui
   ```
   
4. 安装 [x-cli](./docs/product/x-cli)
   
4. 通过 x-cli , 自动链接模块

   ```
   x-cli module autolink 
   ```
