## 里程碑输出

### app-b / app-c

#### 测试包

每完成一个业务模块，会提供一个测试的包。

统一为蒲公英的测试下载地址。（iOS  测试人员需要提供 UUID）

#### 上线

- iOS 应用： 开发人员提交到 app store，由运营人员上线
- android 应用： 开发人员将生产 apk 给到运营人员，由运营人员上线

### 小程序

#### 测试

测试人员提供，微信 id 号，开发人员在完成业务模块后，将体验测试版给到测试人员（二维码）。

#### 上线

开发人员提交小程序到微信平台，由运营人员上线
