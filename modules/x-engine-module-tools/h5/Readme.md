## unZipFile



	
**参数说明**

| name                        | type      | optional | default   | comment  |
| --------------------------- | --------- | -------- | --------- |--------- |
| filePath | string |  |  | 文件路径 |
| unZipPath | string | true |  | 解压到 路径 |
| \_\_event\_\_ | string | true |  | 压缩进度 |


## zipFile



	
**参数说明**

| name                        | type      | optional | default   | comment  |
| --------------------------- | --------- | -------- | --------- |--------- |
| filePath | string |  |  | 文件路径 |
| toZipPath | string | true |  | 压缩到 路径 |
| \_\_event\_\_ | string | true |  | 解压缩进度 |
